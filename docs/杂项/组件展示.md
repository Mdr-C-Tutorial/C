## Quiz 小测验

分为单选和多选；单选选择之后直接给出答案，多选点击“确认答案”之后给出答案。

<Quiz
  type="single"
  :options="[
    { label: 'int p = &a;', value: 'A' },
    { label: 'int *p = a;', value: 'B' },
    { label: 'int *p = &a;', value: 'C' },
    { label: '*int p = &a;', value: 'D' }
  ]"
  answer="C"
>

  <template #question>

若有 `int a = 10;`，如何定义指向 `a` 的指针？

  </template>

  <!-- 这里是默认插槽，用于写解析 -->
  正确答案是 **C**。
  
* **A**: `int p` 只是定义一个整数，不能存地址。
* **B**: `int *p` 需要地址，但 `a` 是值。
* **C**: 正确，`&a` 取出了 `a` 的地址赋值给指针 `p`。

</Quiz>

<Quiz
  type="multi"
  :options="[
    { label: 'int 2number;', value: 'A' },
    { label: 'int _count;', value: 'B' },
    { label: 'int total-sum;', value: 'C' },
    { label: 'int number2;', value: 'D' },
    { label: 'int void;', value: 'E' }
  ]"
  :answer="['B', 'D']"
>
  <template #question>

以下哪些是 C 语言中**合法**的标识符？（多选）

  </template>

  C 语言标识符规则：

  1. 不能以数字开头（排除 A）。
  2. 不能包含减号等特殊符号（排除 C）。
  3. 不能是关键字（排除 E，`void` 是关键字）。
  
  所以正确的是 `_count` 和 `number2`。

</Quiz>
