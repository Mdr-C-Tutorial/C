# 10902

## 题目

解释以下代码中：哪些名字是对象？哪些名字是类型？哪些表达式求值会产生未定义行为？

```c
int main(void) {
    int x;
    int* p = &x;
    return *p;
}
```

## 解析

`x` 是一个 `int` 对象，但它没有被初始化。`p` 是一个指针对象，初始化为 `x` 的地址。

## 答案

对象：

1. `x`；
2. `p`。

类型：

1. `int`；
2. `int*`；
3. `void` 作为函数形参列表中的类型说明。

会产生未定义行为的是 `*p` 的求值。虽然 `p` 指向 `x`，但是 `x` 的值未初始化；读取未初始化的自动存储期 `int` 对象会产生未定义行为。
