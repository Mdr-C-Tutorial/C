# 10201

## 题目

创建 C 语言项目，把本章的示例代码复制进你的项目中，并且进行编译运行。

## 解析

这一题的目标是确认“编辑、编译、运行”这一整条流程能够走通。只要程序能够成功编译，并输出示例代码对应的内容，就完成了本题。

## 答案

可以按本章示例建立一个 `.c` 文件，例如：

```c
#include <assert.h>
#include <stdio.h>

int main(void) {
    puts("Hello, World!");

    int a, b;
    scanf("%d%d", &a, &b);
    assert(b != 0);
    printf("%g\n", (double)a / b);

    return 0;
}
```

如果使用 GCC，可以在终端中运行：

```sh
gcc main.c -o main
./main
```
