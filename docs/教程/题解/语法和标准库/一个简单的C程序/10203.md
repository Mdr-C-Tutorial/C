# 10203

## 题目

实验与观察：探索整数除法。

1. 暂时移除代码中的 `(double)`，让除法变成 `a / b`。
2. 运行程序，分别输入 `5 2`、`6 3`、`-5 2`，观察并记录输出结果。
3. 尝试解释为什么结果是这样。这是否验证了教程中关于“整数除法向零取整”的说法？

## 解析

当 `/` 的两个操作数都是整数类型时，执行的是整数除法。整数除法会舍弃小数部分，并向零取整。

## 答案

把表达式改成 `a / b` 后，三个输入的典型结果如下：

```txt
5 2   -> 2
6 3   -> 2
-5 2  -> -2
```

`5 / 2` 的数学结果是 `2.5`，整数除法得到 `2`；`-5 / 2` 的数学结果是 `-2.5`，向零取整得到 `-2`。这验证了本章关于整数除法的说明。
