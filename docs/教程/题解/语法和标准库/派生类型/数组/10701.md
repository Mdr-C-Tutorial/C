# 10701

## 题目

输入两个整数 m 和 n（均小于 100），代表矩阵的行数和列数。接着输入两个 m x n 的整数矩阵 A 和 B。计算它们的和矩阵 C = A + B，并输出矩阵 C。

## 解析

对应位置直接相加。

## 答案

```c
#include <stdio.h>
#include <stdlib.h>

int main() {
    int rows, cols;
    int **matrix_a, **matrix_b, **matrix_c;
    int i, j;

    printf("输入行数和列数：");
    scanf("%d %d", &rows, &cols);

    matrix_a = (int **)malloc(rows * sizeof(int *));
    matrix_b = (int **)malloc(rows * sizeof(int *));
    matrix_c = (int **)malloc(rows * sizeof(int *));

    for (i = 0; i < rows; i++) {
        matrix_a[i] = (int *)malloc(cols * sizeof(int));
        matrix_b[i] = (int *)malloc(cols * sizeof(int));
        matrix_c[i] = (int *)malloc(cols * sizeof(int));
    }

    printf("输入矩阵 A 的元素:\n");
    for (i = 0; i < rows; i++) {
        for (j = 0; j < cols; j++) {
            scanf("%d", &matrix_a[i][j]);
        }
    }

    printf("输入矩阵 B 的元素:\n");
    for (i = 0; i < rows; i++) {
        for (j = 0; j < cols; j++) {
            scanf("%d", &matrix_b[i][j]);
        }
    }

    for (i = 0; i < rows; i++) {
        for (j = 0; j < cols; j++) {
            matrix_c[i][j] = matrix_a[i][j] + matrix_b[i][j];
        }
    }

    printf("和矩阵 C:\n");
    for (i = 0; i < rows; i++) {
        for (j = 0; j < cols; j++) {
            printf("%d ", matrix_c[i][j]);
        }
        printf("\n");
    }

    for (i = 0; i < rows; i++) {
        free(matrix_a[i]);
        free(matrix_b[i]);
        free(matrix_c[i]);
    }
    free(matrix_a);
    free(matrix_b);
    free(matrix_c);

    return 0;
}
```
