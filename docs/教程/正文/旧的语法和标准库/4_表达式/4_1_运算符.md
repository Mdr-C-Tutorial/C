# 运算符

C 用**运算符**表示算术运算
几种数学中常见的运算符号在 C 中有相似的表示

C 中有 30 多种运算符，此章节只介绍一部分，并且只介绍它们的操作数都是算术类型时的行为

## 1. 操作数

对于一个运算符来说，需要传入的对象叫做**操作数**，需要操作数数量为 **n** 的运算符称为 **n 元运算符**

#### 2.2.5.2 除法 `/`

1. 整数除以整数，结果为整数，向零取整：

   ```c
   int a = 6 / 3;  // a = 2
   int b = 5 / 3;  // b = 1
   int c = -6 / 2; // c = -3
   int d = -5 / 2; // d = -2
   ```

#### 2.2.5.3 取余 `%`

   对于上面的式子，`/` 得到商，`%` 得到余数

## 3. 运算符宏

> 在头文件 `<iso646.h>` 中定义

| 运算符    | 宏        |
|--------|----------|
| `&&`   | `eq`     |
| `&=`   | `and_eq` |
| `&`    | `bitand` |
| `\|`   | `bitor`  |
| `~`    | `compl`  |
| `!`    | `not`    |
| `!=`   | `not_eq` |
| `\|\|` | `or`     |
| `\|=`  | `or_eq`  |
| `^`    | `xor`    |
| `^=`   | `xor_eq` |
