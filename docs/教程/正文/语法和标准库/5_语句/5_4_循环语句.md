# 循环语句

循环语句用于表达“在条件满足时重复执行”。本节作为总览，先建立三种循环形式的选择直觉，再进入子章节细节。

## 1. 三种循环的定位

`while` 适合“先判断再执行”；`do while` 适合“至少执行一次”；`for` 适合把初始化、条件和步进写在同一位置。三者在能力上可互相改写，但在可读性上各有优势。

## 3. 阅读建议

阅读循环语句时，优先盯住三件事：终止条件是否可达、每轮状态是否有推进、异常路径是否会破坏收尾动作。只要这三点清楚，循环控制流通常就可靠。
