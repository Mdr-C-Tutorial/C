# C 时间日期库

C 语言提供了 `time.h` 头文件来处理日期和时间。这个库包含了处理时间的函数和类型定义。

## 重要的时间类型

### 日历时间 `time_t`

`time_t` 是一个用于存储时间的基本类型，通常是一个长整型，表示从 1970 年 1 月 1 日 00:00:00 UTC（称为 UNIX 纪元）开始经过的秒数。

这里需要提一嘴：上面这个“UNIX 纪元”描述是常见实现习惯，不是 ISO C 的强制定义；标准只要求 `time_t` 能表达日历时间。

### 分解时间 `struct tm`

`struct tm` 是一个用于保存日期和时间信息的结构体，包含以下成员：

```c
struct tm {
    int tm_sec;   // 秒 (0-59)
    int tm_min;   // 分 (0-59)
    int tm_hour;  // 时 (0-23)
    int tm_mday;  // 日 (1-31)
    int tm_mon;   // 月 (0-11，0 表示一月)
    int tm_year;  // 年 (从 1900 年开始的年数)
    int tm_wday;  // 星期几 (0-6，0 表示星期日)
    int tm_yday;  // 一年中的第几天 (0-365)
    int tm_isdst; // 夏令时标志
};
```

运行结果：该代码块主要用于语法或结构说明，单独运行通常无终端输出。

### 使用示例

```c
#include <stdio.h>
#include <time.h>

int main() {
    // 获取当前时间
    time_t now;
    time(&now);

    // 转换为本地时间
    struct tm *local = localtime(&now);

    // 使用 strftime 格式化输出
    char str[80];
    strftime(str, sizeof(str), "%Y-%m-%d %H:%M:%S", local);
    printf("当前时间：%s\n", str);

    // 使用 ctime 直接输出
    // printf("ctime 输出：%s", ctime(&now));

    return 0;
}
```

### 可重入 (Reentrant) 的时间函数

`ctime()`、`asctime()`、`localtime()`、`gmtime()` 返回的指针通常指向静态存储区，因此并发场景下需要额外同步。`ctime()` 与 `asctime()` 在 C23 中已被弃用，推荐统一使用 `strftime()` 做格式化输出。

这里需要提一嘴：`localtime_r()` 与 `gmtime_r()` 属于 POSIX 扩展，不是 ISO C 必选接口；若要写严格 ISO C 代码，应优先使用标准接口并自行管理并发访问策略。

```c
#include <stdio.h>
#include <time.h>

int main(void) {
    time_t now = time(NULL);
    struct tm *tm_ptr = localtime(&now);
    char buf[64];

    if (tm_ptr == NULL) {
        return 1;
    }

    if (strftime(buf, sizeof(buf), "%Y-%m-%d %H:%M:%S", tm_ptr) == 0) {
        return 1;
    }

    puts(buf);
    return 0;
}
```

## 计时函数

### CPU 时间 `clock()`

`clock()` 返回进程消耗的处理器时间刻度值，单位不是秒，而是“时钟滴答”。要换算为秒，需要除以 `CLOCKS_PER_SEC`。

```c
clock_t start = clock();
/* ... 计算任务 ... */
clock_t end = clock();
double cpu_seconds = (double)(end - start) / CLOCKS_PER_SEC;
```

运行结果：该代码块主要用于语法或结构说明，单独运行通常无终端输出。


它适合粗粒度性能估计，不适合做高精度墙钟时间测量。

### 宏 `CLOCKS_PER_SEC`

这个宏给出 `clock()` 结果转秒的换算因子。它是编译期常量，不同实现可能取不同值，因此不应手写假定值。

## 高精度时间测量函数

C 语言中提供了一种结构体 `struct timespec`，用于表示高精度（精确到纳秒的）时间。

```c
struct timespec {
    time_t tv_sec; // 秒
    long tv_nsec; // 纳秒
};
```

运行结果：该代码块主要用于语法或结构说明，单独运行通常无终端输出。


### 获取高精度当前时间

ISO C 标准接口是 `timespec_get`（C11 起）：

```c
int timespec_get(struct timespec *ts, int base);
```

运行结果：该代码块主要用于语法或结构说明，单独运行通常无终端输出。


当 `base == TIME_UTC` 且函数返回 `TIME_UTC` 时，`ts` 中写入当前 UTC 时间。

```c
struct timespec ts;
if (timespec_get(&ts, TIME_UTC) == TIME_UTC) {
    /* ts.tv_sec 与 ts.tv_nsec 可用 */
}
```

## 时间计算

### 计算时间差

```c
double difftime(time_t time2, time_t time1);  // 返回两个时间之间的秒数差
```

运行结果：该代码块主要用于语法或结构说明，单独运行通常无终端输出。


`difftime(time2, time1)` 的语义是 `time2 - time1`，返回值类型是 `double`。当 `time_t` 不是简单整数类型时，`difftime` 比直接相减更稳妥。

### 时间操作示例

```c
#include <stdio.h>
#include <time.h>

int main() {
    time_t start, end;

    time(&start);
    // 执行一些操作
    time(&end);

    double diff = difftime(end, start);
    printf("耗时：%f 秒\n", diff);

    return 0;
}
```
