# 注释

## 1. 定义

注释的作用是在代码内写文档供人类阅读；编译器会忽略注释（技术上，是把注释替换成一个空格）。

## 2. 分类

1. **单行注释**：单行注释以 `//` 开始：编译器会忽略从 `//` 开始到行末的所有字符。
2. **多行注释**：多行注释以 `/*` 开始，以 `*/` 结束。编译器会忽略从 `/*` 开始到 `*/` 结束的所有字符，包括 `/*` 和 `*/`。

## 3. 示例

```c
#include <stdio.h>
int main(void) {
    // 这是单行注释
    /* 这是多行注释 */
    puts("Hello World 01");
    // puts("Hello World 02");

    /*
      puts("Hello World 03");
      puts("Hello World 04");
      puts("Hello World 05");
    */

    /*
     * 为了整齐美观，也可以这样写：
     * puts("Hello World 06");
     * puts("Hello World 07");
     * puts("Hello World 08");
     */
    return 0;
}
```

可能的输出：

```ansi
Hello World 01
```
