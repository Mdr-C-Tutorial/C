# 对象

本章围绕 C 语言的一个核心概念：对象 (Object)。

在 C 语言里：

1. **对象**是执行环境中一块用于存储的区域；
2. 对象有**类型** (Type)、**大小** (Size) 与**对齐要求** (Alignment requirement)；
3. 对象在其**生存期** (Lifetime) 内，承载某个**值** (Value)（或承载不确定值 (Indeterminate value)）。

本章的目标是把“对象是什么、如何表示、如何被访问、何时存在”讲清楚，并把它们和声明语法、类型限定符、链接与存储期等规则关联起来。

```c
int main(void) {
    int x;
    int* p = &x;
    return *p;
}
```
