# 常量和字面量

本节讨论三类内容：

1. **常量** (Constant)：一个值（或表达式结果）在某个语境中被要求“编译期可确定”；
2. **字面量** (Literal)：直接写在源代码中的常量记号，例如 `123`、`3.14`、`'A'`、`"abc"`；
3. **布尔常量**：`true` / `false` 及其标准定义方式。
