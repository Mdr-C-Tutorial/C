# 运算符

运算符 (Operator) 用于把一个或多个操作数 (Operand) 组合成一个表达式。

本章的关键点不在于“记住所有符号”，而在于掌握三件事：

1. 每个运算符对操作数的**类型要求**；
2. 运算符在求值时会发生的**隐式转换**；
3. 运算符与副作用相关的**求值顺序/序列规则**。
