# 求值和副作用

本节给出两个正式但易用的定义。

## 1. 值计算

值计算 (Value computation) 指的是：为了得到一个表达式的结果值而进行的计算过程。

例如：`3 + 2` 的值计算会产生结果值 `5`。

## 2. 副作用

副作用 (Side effect) 指的是：表达式求值过程中，对程序状态造成的可观察改变。

常见副作用包括：

1. 修改对象：例如 `i++`、`a = 1`。
2. I/O：例如调用 `printf` 输出。
3. 调用函数（函数体内可能进一步产生副作用）。
