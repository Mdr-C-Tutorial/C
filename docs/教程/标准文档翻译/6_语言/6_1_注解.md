# 6.1 注解

> [!TIP]
> 这一节实际上是在说明本章语法记号怎么读。后面看到斜体、粗体、冒号、`opt` 下标时，都应回到这里理解。

[[toc]]

1. 本章使用的语法记号中，语法类别（非终结符）用斜体表示，字面单词和字符集成员（终结符）用粗体表示。非终结符后面的冒号（`:`）用于引出其定义。不同的可选定义列在不同行上；但若前面写有 “one of”，则例外。可选符号用下标 `opt` 表示，因此：

```text
{ expressionopt }
```

表示一个被花括号包围的可选表达式。

2. 当正文中提到这些语法类别时，它们不再使用斜体，并且单词之间用空格分隔，而不是用连字符连接。

3. 语言语法的汇总见附录 A。
