# 6.5.18 逗号运算符

[[toc]]

1. 逗号运算符表达式的形式是：

```c
E1, E2
```

2. 左操作数会先按 `void` 表达式求值，也就是只保留其副作用，不保留其值。

3. 在左操作数求值与右操作数求值之间，存在一个序列点。

4. 然后求值右操作数；整个逗号运算符表达式的类型和值，都取自右操作数。

5. 逗号运算符结果不是左值。

6. 这里说的是**逗号运算符**，不是函数实参列表、初始化器列表等语境里用来分隔项目的逗号。

::: details 标准示例
```c
f(a, (t = 3, t + 2), c)
```

这里函数仍然只有三个实参；第二个实参是逗号运算符表达式，其值为 `5`。
:::
