# 6.5.6 乘法运算符

[[toc]]

1. 乘法表达式包括：
   - `E1 * E2`
   - `E1 / E2`
   - `E1 % E2`

2. `*` 与 `/` 的两个操作数都必须具有算术类型；`%` 的两个操作数必须具有整数类型。

3. 若任一操作数是十进制浮点类型，则另一操作数不得是标准浮点类型、复数类型或虚数类型。

4. 对操作数先执行通常算术转换。

5. `E1 * E2` 的结果是乘积。

6. `E1 / E2` 的结果是商，`E1 % E2` 的结果是余数。

7. 若第二个操作数为 `0`，则 `/` 和 `%` 的行为都未定义。

8. 对整数除法，`/` 的结果是代数商去掉小数部分，也就是朝零截断。

9. 若商 `a / b` 可表示，则必须满足：

```text
(a / b) * b + a % b == a
```

否则，`a / b` 与 `a % b` 的行为都未定义。

> [!NOTE]
> 这里的整数除法不是向下取整，而是朝零截断。所以负数参与除法时，结果常常与一些脚本语言不同。
