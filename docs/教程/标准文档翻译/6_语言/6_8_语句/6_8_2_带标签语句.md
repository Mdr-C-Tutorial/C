# 6.8.2 带标签语句

## 语法

::: details 形式化语法

```bnf
label:
    attribute-specifier-sequenceopt identifier :
    attribute-specifier-sequenceopt case constant-expression :
    attribute-specifier-sequenceopt default :

labeled-statement:
    label statement
```

:::

## 约束

1. `case` 标签和 `default` 标签只能出现在 `switch` 语句中。与这两类标签有关的进一步约束，见 `switch` 语句。
2. 标签名在同一函数内必须唯一。

## 语义

1. 复合语句中的任意语句或声明，都可以前置一个前缀，把某个标识符声明为标签名。
2. 可选的属性说明符序列归属于该标签。
3. 标签本身不改变控制流；控制流会直接越过标签继续执行。
