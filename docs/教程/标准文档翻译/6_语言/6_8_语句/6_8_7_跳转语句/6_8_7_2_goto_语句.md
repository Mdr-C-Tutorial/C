# 6.8.7.2 goto 语句

## 约束

1. `goto` 语句中的标识符必须命名其外围函数中某处定义的一个标签。
2. `goto` 语句不得从某个具有变长修改类型的标识符作用域之外，跳转到该标识符作用域之内。

## 语义

1. `goto` 语句会无条件跳转到外围函数中由该标签前缀标记的语句。

## 示例

### 示例 1

有时跳入一大段复杂语句的中间位置会更方便。下面的结构体现了一种典型做法：

```c
/* ... */
goto first_time;
for (;;) {
    // 确定下一步操作
    /* ... */
    if (need to reinitialize) {
        // 仅重新初始化的代码
        /* ... */
    first_time:
        // 通用初始化代码
        /* ... */
        continue;
    }
    // 处理其他操作
    /* ... */
}
```

这个写法通常建立在以下假设上：

1. 通用初始化代码只访问当前函数可见的对象。
2. 通用初始化代码过大，不值得复制。
3. 用于确定下一步操作的代码位于循环头部，以便 `continue` 等语句能够重新到达它。

### 示例 2

跳过带有变长修改类型对象声明的作用域边界是不符合要求的；但在该作用域内部跳转则是有效的：

```c
goto lab3;                 // 无效：跳入 VLA 的作用域
{
    double a[n];
    a[j] = 4.4;
lab3:
    a[j] = 3.3;
    goto lab4;             // 有效：在 VLA 作用域内部跳转
    a[j] = 5.5;
lab4:
    a[j] = 6.6;
}
goto lab4;                 // 无效：跳入 VLA 的作用域
```
