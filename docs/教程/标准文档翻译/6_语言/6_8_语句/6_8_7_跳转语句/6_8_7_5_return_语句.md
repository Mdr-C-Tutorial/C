# 6.8.7.5 return 语句

## 约束

1. 带表达式的 `return` 语句不得出现在返回类型为 `void` 的函数中。
2. 不带表达式的 `return` 语句只能出现在返回类型为 `void` 的函数中。

## 语义

1. `return` 语句会终止当前函数的执行，并把控制权返回给调用者。
2. 一个函数可以有任意多个 `return` 语句。
3. 如果执行的是带表达式的 `return` 语句，那么该表达式的值会作为函数调用表达式的值返回给调用者。
4. 如果该表达式的类型与其所在函数的返回类型不同，那么这个值会像赋值给一个具有该函数返回类型的对象那样进行转换。

## 示例

```c
struct s { double i; } f(void);
union {
    struct {
        int f1;
        struct s f2;
    } u1;
    struct {
        struct s f3;
        int f4;
    } u2;
} g;

struct s f(void)
{
    return g.u1.f2;
}

/* ... */
g.u2.f3 = f();
```

这里不存在未定义行为；但如果不通过函数调用取值，而是直接做相应赋值，则会产生未定义行为。
