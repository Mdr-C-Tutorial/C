# 6.8.5.3 switch 语句

## 约束

1. `switch` 语句的控制表达式必须具有整数类型。
2. 如果某个 `switch` 语句关联的 `case` 或 `default` 标签位于一个具有变长修改类型的标识符作用域内，那么整个 `switch` 语句都必须位于该标识符的作用域内。
3. 每个 `case` 标签中的表达式都必须是整数常量表达式。
4. 同一个 `switch` 语句关联的任意两个 `case` 常量表达式，在转换后都不得具有相同值。
5. 与同一个 `switch` 语句关联的 `default` 标签最多只能有一个。
6. 被包含的 `switch` 语句可以有自己的 `default` 标签，也可以有与外层 `switch` 重复的 `case` 值；上述唯一性要求只针对同一个 `switch`。

## 语义

1. `switch` 语句会根据控制表达式的值，以及 `switch` 体上或体内 `case` 标签的值、`default` 标签是否存在，使控制流跳转到 `switch` 体中的某处、进入其中，或者越过整个 `switch` 体。
2. `case` 标签和 `default` 标签只能在最近包围它们的那个 `switch` 语句中可达。
3. 对控制表达式先执行整数提升。
4. 每个 `case` 标签里的常量表达式，都会转换为控制表达式提升后的类型。
5. 若某个转换后的 `case` 值与提升后的控制表达式值相匹配，控制流就跳转到该 `case` 标签后面的语句或声明。
6. 否则，如果存在 `default` 标签，控制流就跳转到 `default` 标签后面的语句或声明。
7. 如果没有任何 `case` 匹配，且也没有 `default` 标签，那么 `switch` 体中的任何部分都不会执行。

## 实现限度

1. 如 [5.2.5.2 环境限度](/教程/标准文档翻译/5_环境/5_2_5_环境限度) 所述，实现可以限制一个 `switch` 语句中 `case` 值的数量。

## 示例

```c
switch (expr)
{
    int i = 4;
    f(i);
case 0:
    i = 17;
    /* 落入 default 分支 */
default:
    printf("%d\n", i);
}
```

上面的对象 `i` 在该块内具有自动存储期，但它永远不会完成初始化；因此，如果控制表达式的值不是 `0`，那么 `printf` 会读取一个具有不确定表示的对象。类似地，对函数 `f` 的调用也不可能到达。
