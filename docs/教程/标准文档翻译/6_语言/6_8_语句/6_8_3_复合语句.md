# 6.8.3 复合语句

## 语法

::: details 形式化语法

```bnf
compound-statement:
    { block-item-listopt }

block-item-list:
    block-item
    block-item-list block-item

block-item:
    declaration
    unlabeled-statement
    label
```

:::

## 语义

1. 如果一个复合语句是函数体，那么它与参数类型列表，以及二者之间可选的属性说明符序列，一起构成该函数定义所关联的块。
2. 在其他情形下，复合语句构成一个与任何其他块都不同的独立块。
3. 标签在翻译时，应视为其后跟着一个空语句。
