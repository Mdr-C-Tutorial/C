# 1. 涵盖范围

1. 本文档规定了用 C 程序设计语言编写的程序的形式，并确立了对这些程序的解释规则。其目的是促进 C 语言程序在各种数据处理系统中的可移植性。本文档旨在供（语言）实现者和程序员使用。它规定了：
    * C 程序的表示；
    * C 语言的语法和约束；
    * 解释 C 程序的语义规则；
    * 由 C 程序处理的输入数据的表示；
    * 由 C 程序产生的输出数据的表示；
    * 由符合标准的 C 实现所施加的限制和限度。

2. 本文档不规定：
    * 将 C 程序转换为供数据处理系统使用的机制；
    * 由数据处理系统调用 C 程序以供使用的机制；
    * 将输入数据转换为供 C 程序使用的机制；
    * 输出数据在由 C 程序产生后被转换的机制；
    * 超出任何特定数据处理系统容量或特定处理器容量的程序及其数据的大小或复杂度；
    * 能够支持符合标准的实现的数据处理系统的所有最低要求。

3. 附录 J 概述了 C 程序可能遇到的可移植性问题。
