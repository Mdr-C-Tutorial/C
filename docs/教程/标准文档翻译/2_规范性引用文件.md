# 2. 规范性引用文件

> [!NOTE]
> “规范性引用文件”不是参考阅读材料，而是正文通过引用把它们的一部分内容纳入了本标准要求范围。凡正文以这种方式引用，相关内容就具有规范效力。

1. 下列文件在本文中被以某种方式引用，使得其部分或全部内容构成本文档的要求。对于注明日期的引用文件，仅所引用的版本适用。对于未注明日期的引用文件，适用其最新版本（包括任何修订）。

2. ISO/IEC 2382:2015，_Information technology — Vocabulary_（信息技术词汇）。

3. ISO 4217，_Codes for the representation of currencies_（货币表示代码）。

4. ISO 8601 系列，_Data elements and interchange formats — Information interchange — Representation of dates and times_（数据元和交换格式 信息交换 日期和时间的表示）。

5. ISO/IEC 10646，_Information technology — Universal Coded Character Set (UCS)_（信息技术通用编码字符集（UCS））。

6. ISO/IEC 60559:2020，_Information technology — Microprocessor Systems — Floating-Point arithmetic_（信息技术 微处理器系统 浮点运算）。

7. ISO 80000-2，_Quantities and units — Part 2: Mathematics_（量和单位 第 2 部分：数学）。

8. Unicode Consortium（Unicode 联盟）。_Unicode Standard Annex, UAX #44, Unicode Character Database [online]_（Unicode 标准附录 UAX #44，Unicode 字符数据库 [在线]）。编辑：Ken Whistler。可从 <https://www.unicode.org/reports/tr44> 获取。

9. Unicode Consortium（Unicode 联盟）。_The Unicode Standard, Derived Core Properties_（Unicode 标准：派生核心属性）。可从 <https://www.unicode.org/Public/UCD/latest/ucd/DerivedCoreProperties.txt> 获取。
